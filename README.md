# d5



# 前置条件

已root分辨率为1920*1080安卓机

## 使用说明

准备阶段

1. 下载按键精灵手机助手
2. 复制d5.txt内容至【脚本】
3. 复制ui.txt内容至【界面】
4. 将attachments文件夹下内容上传至附件

----------------

1. 注册按键账号
2. 登录后台创建应用


发布阶段

1. 点击【发布】->【制作按键小精灵】
2. 输入【小精灵名称】
3. 输入【小精灵包名】
4. 【选择应用】
5. 【选择分辨率】
6. 生成ROOT版

使用阶段

1. 安装完app后开启应用权限【打开悬浮窗】
